## Zombie Car Game – Unity Technical Test
Overview

This project is a portrait-mode arcade driving game built in Unity where the player drives a car inside an arena and earns points by hitting zombies within a limited time.

The goal is to demonstrate basic gameplay systems, physics interaction, UI management, and mobile-friendly controls using Unity.

Gameplay

The player controls a car that moves around a closed arena. Zombies spawn randomly across the arena and the player must collide with them to score points.

When a zombie is hit:

The score increases

The zombie reacts using physics forces

The zombie is destroyed after a short delay

The game runs under a countdown timer, and once the timer reaches zero the round ends and the final score is displayed.

Features

• Car driving mechanics using Prometeo Car Controller
• Zombie spawning system
• Physics-based zombie collision reaction
• Score tracking system
• Countdown timer
• Game Over screen
• Restart functionality
• Android build support

The car uses simplified mobile controls suitable for portrait gameplay.

Game Loop

1. Game starts and timer begins counting down.

2. Zombies spawn randomly in the arena.

3. Player drives the car and collides with zombies.

4. Each zombie hit increases the score.

5. Zombies react with physics and are removed after a delay.

6. When the timer reaches 0, the round ends.

7. Player can restart the game.
